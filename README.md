# AI Knowledge Hub
基于 Spring AI Alibaba + Ollama + ElasticSearch + MinIO + Redis 的私有部署 RAG 智能知识库问答系统

## 项目介绍
AI Knowledge Hub 是一款轻量级、可私有化部署的企业级 AI 知识库平台。
通过本地大模型 + RAG 检索增强生成技术，实现文档上传、自动切片、向量检索、智能问答。

## 技术栈
- 后端：Spring Boot 3.x
- AI 框架：Spring AI Alibaba
- 本地大模型：Ollama
- 向量检索：ElasticSearch
- 文件存储：MinIO
- 缓存：Redis
- 核心技术：RAG（检索增强生成）

## 核心功能
- 📄 多格式文档上传（PDF / Word / TXT ）
- 🧠 自动文档分块、清洗、向量化
- 🔍 基于向量的语义检索
- 🤖 本地大模型智能问答
- ☁ 私有化部署，数据安全可控
- 📊 文档管理 + 对话历史管理

## 适用场景
- 企业内部知识库
- 私有文档智能问答
- 法律/金融/学术文献检索
- 个人本地 AI 助手
